How to Use

  1. Download and Install:

    Download the appropriate version for your operating system and install it.

  2. Enter API Key:

    Enter your Groq API key in the settings panel.

  3. Select Model and Language:

    Choose your preferred model and language for transcription.

  4. Start Recording:

    Click the microphone button to start recording your speech.

  5. View Transcription:

    Your speech will be transcribed in real-time and displayed in the app.

  6. Copy Text:

    Click the copy button to copy the transcribed text to your clipboard.

Requirements

  • A Groq API key (sign up at groq.com)
  • Microphone access
  • Internet connection
