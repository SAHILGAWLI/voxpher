Getting Started with Voxpher

Follow these simple steps to set up your application after installation.

1

Visit Groq Cloud

Go to Groq Cloud to get your API key. Click here to visit

2

Sign In and Create Key

Sign in using your Google account, then click on "Create API Key" in the top right corner.

3

Name and Copy Key

Name your key anything you want and copy the generated API key. Note: This key will be shown only once!

4

Run Voxpher

Launch the Voxpher application you've just installed.

5

Access Settings

Double click on the microphone icon in the sidebar to access the settings panel.

Application Settings

6

Set API Key and Save

Paste your Groq API key under "GROQ API KEY" field, click "Save Settings" and you're ready to go!
